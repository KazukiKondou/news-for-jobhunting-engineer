#技術トレンド #OSS #Homebrew

Homebrew 6.0.0公開、サードパーティーtapに「Tap Trust」確認を必須化 — macOS 27 Golden Gate初期対応、Intel x86_64は2027年9月に完全終了

macOS / Linuxのパッケージマネージャ「Homebrew」のメジャーバージョン6.0.0が2026年6月11日に公開された。最大の変更は新セキュリティ機構「Tap Trust」で、サードパーティーtap(formula や cask が入ったGitリポジトリ)はインストール前に明示的な信頼登録が必要になる。tap はRubyスクリプトでユーザー権限のまま走るため、これまで悪意あるtapを区別できなかったサプライチェーンの穴を塞いだ形。brew trust --formula user/repo/formula や brew trust --cask user/repo/cask で個別許可、Brewfileは trusted: true を付記する。あわせてmacOS 27「Golden Gate」への初期対応、LinuxのBubblewrapサンドボックス対応、ask mode標準化、内部JSON APIの標準有効化も入った。Intel Macは2026年9月にTier 3へ、macOS Intel x86_64の完全サポート終了は2027年9月と予告された。

#技術トレンド #AIエージェント #OSS

Xiaomi、Claude Code的なAIエージェント「MiMo Code」をMITで公開 — 100万トークン文脈、200ステップ超で人間評価65%勝率

中国Xiaomiが2026年6月11日、AIコーディングエージェント「MiMo Code」と背後のモデル「MiMo-V2.5」「MiMo-V2.5-Pro」をMITライセンスで公開した。OpenCodeをフォークし、複数エージェント候補生成と評価選別、自然言語SKILLをJavaScript生成で決定論的に実行する仕組みを備える。コンテキストウィンドウは100万トークン、200ステップを超える長時間タスクではブラインドの人間評価で MiMo-V2.5 が65%の勝率を記録した。GitHubは github.com/XiaomiMiMo/MiMo-Code、ドキュメントは mimo.xiaomi.com/mimocode/start でいずれもセルフホスト可能。Claude Code / Codex / Gemini Code に続く中華圏発のOSSコーディングエージェントとして、選択肢の多様化が進む。

#企業動向 #OpenAI #外資テック

OpenAI、クラウド開発環境のOna(旧Gitpod)を買収 — Codex週500万人ユーザーの長時間タスク基盤へ

OpenAIが米Onaの買収を2026年6月11日(現地時間)に発表した(買収額は非公表)。Onaは旧Gitpodで、Webブラウザベースのクラウド開発環境を200万人超の開発者に提供してきた老舗。2025年9月のリブランド以降は「AIエージェント向けのセキュアなクラウド実行・オーケストレーション基盤」へ業態転換し、銀行・政府系ファンドでの週次利用は2026年初以降で約13倍に伸びていた。OpenAIは買収理由として「Codexの作業時間が数時間〜数日に伸びており、ユーザーが単一デバイス・セッションに縛られない長時間実行環境が必要」と説明。Codexの週次アクティブユーザーは500万人超で、2026年初比+400%。規制当局承認後、OnaチームはCodexチームに合流する。

#企業動向 #Anthropic #外資テック

Anthropic、Claude Fable 5の不可視セーフガード問題を公式謝罪 — Claude Opus 4.8への自動フォールバックへ移行

Anthropicが2026年6月11日、AIモデル「Claude Fable 5」にユーザーへ通知せず仕込んでいたセーフガード(モデル性能の不可視制限)について公式に謝罪する声明をXで公開した。「Cancer」「Hello」など無害な入力でも危険判定される事例が報告されており、全セッションのうち5%が誤って危険判定される過剰な検知になっていた。Anthropicは「迅速かつ安全に展開するため不可視セーフガードを選択したが、それは誤りだった」と表明。今後はフラグ付きリクエストを Claude Opus 4.8 へ明示的にフォールバックする処理に切り替え、サイバー・生物兵器関連と同じ拒否フローへ統一する。API経由では拒否理由を明示し、分類器改善で誤判定頻度を引き下げる予定。フロンティアAIの「目に見えない制御」をめぐる議論が一段表に出た出来事。

NHK Eテレ『100分de名著』の2026年6月放送回は、2024年に92歳で逝去した詩人・谷川俊太郎の自選詩集を、批評家・若松英輔が「25分×4回=100分」で読み解く構成。テキストはNHK出版から5月25日に発売(全153ページ)。詩は短いがゆえに、ことばの一語一語の選択がそのまま書き手の思考の輪郭を露わにする。AI時代に「自分のことばで何かを言う」とはどういうことか、書き手の選択と削除の過程を追体験することで、自分自身の思考のクセを観察するメタ認知の訓練として読める。LLMが平均的な文を量産する時代に、「平均的でない一語」を選び抜く作業の重さを再考する糸口になる一冊。
