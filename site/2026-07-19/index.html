#採用 #jig.jp #プロダクト横断

jig.jp、採用サイトを全面リニューアル — 「ふわっち」に頼らない複数プロダクト横断の「挑戦のカルチャー」を訴求、東証グロース5244

株式会社jig.jp(東証グロース 5244、本社福井県鯖江市、代表 川俣大)は2026年7月16日、採用サイトを全面リニューアルしたと発表した。ライブ配信「ふわっち」の認知は高い一方で「1プロダクト企業」に見えてしまう課題感から、VTuber事業・スマートグラス「SABERA」・オープンデータ基盤など複数プロダクトを横断する「挑戦のカルチャー」を前面に押し出した構成に変更。人事責任者の桶側是宏氏は「複数のプロダクトで力をつけて挑戦するカルチャーを、1プロダクトのイメージに埋もれさせない」とコメントしている。リニューアル以降、自社サイト経由の直接応募が増加しており、今後は社員の日常や地方拠点の働き方をSNSでも発信するという。福井本社×東京拠点で複数プロダクトを走らせるWeb企業のブランディング事例として、地方拠点型スタートアップの採用戦術の一つを示した。

#nginx #セキュリティ #CVE

nginx が3件の脆弱性を修正、v1.30.4/v1.31.3公開 — CVE-2026-42533 は CVSS 9.2 でDoS/RCE、Web運用は緊急パッチ推奨

Webサーバー「nginx」が2026年7月16日、Stable v1.30.4・Mainline v1.31.3 を公開し、深刻度Critical含む3件の脆弱性を修正した。CVE-2026-42533(CVSS 9.2)はmapディレクティブ処理のヒープバッファオーバーフローで、認証なしでworker再起動によるDoS、ASLR無効環境ではコード実行の恐れがある。CVE-2026-60005(CVSS 8.8)はsliceモジュールの未初期化メモリ利用でメモリ内容の漏洩、CVE-2026-56434(CVSS 8.3)はssiモジュールの Use-After-Free。運用中のWebサービスはStable系v1.30.4またはMainline系v1.31.3への速やかな更新が必要。

#AWS #CloudFront #SRE

AWS CloudFront がグローバル障害、VPC Origins機能に起因 — 17:40発生、PayPay・note・アニメマニアが数時間停止、21:20頃復旧

Amazon Web Services (AWS) の CDN「Amazon CloudFront」が2026年7月16日17時40分頃からグローバル規模で障害を起こし、PayPay・note・アニメマニアなど CloudFront に依存する日本国内のサービスも数時間にわたり停止した。AWSステータスによると、CloudFrontの新機能「VPC Origins」を有効化しているディストリビューションでエラーが発生しており、暫定回避策として同機能を無効化するよう案内が出された。21時20分頃にAWSが復旧アップデートを発表。Web系事業会社の入社1〜2年目が直面するSRE案件で、単一CDNベンダー依存とマルチCDNフェイルオーバーの是非を議論するきっかけになった。

#Google #GeminiSpark #AIエージェント

Google の常時稼働AIエージェント「Gemini Spark」日本語対応 — Ultra 月14,500円、Gmail・カレンダー横断で24時間タスク実行

Google が2026年7月16日、パーソナルAIエージェント「Gemini Spark」の日本語対応を開始した。Gemini 3.5 と Google Antigravity 上で動作し、PCを閉じても・スマホがロック中でも24時間365日クラウド側でタスクを継続実行する。情報収集・データ整理・メール処理・スケジュール調整・旅行プランニングなどをGmail・Docs・Slides横断で自動化。提供対象は「Gemini Ultra」プラン(月額14,500円〜)加入者で、Gemini アプリから順次ロールアウトされる。サードパーティアプリや Model Context Protocol(MCP)経由の任意アプリ連携にも対応。従来のCopilot型AIとの対比で、常時稼働・自律実行型エージェントが月額課金SaaSとしてどう定着するかが焦点となる。

Anthropic、Claude Fable 5 を Max・Team Premium プランの標準機能に統合 — Pro は Usage Credits へ移行、初回$100付与
