#GoogleCloud #PostgreSQL #データベース

Google Cloud、AlloyDBに新ホットスタンバイ — 通常30秒以内のフェイルオーバーとウォームキャッシュを両立

Google Cloudは6月1日、PostgreSQL互換のAlloyDB for PostgreSQLに新たなホットスタンバイ方式を導入したと公表した。従来はスタンバイノードがアイドル待機していたが、新方式では起動済みノードが常時レプリケーションを継続する。プライマリ障害時のフェイルオーバーは通常30秒以内に完了し、メモリキャッシュもウォーム状態のため切り替え直後から本来の処理速度で稼働できる。PostgreSQL 18対応の新規インスタンスから自動適用となり、ミッションクリティカル系の利用シーンで停止時間が桁違いに縮む内容だ。AWS RDS/Aurora側のフェイルオーバー速度競争にも直接効く発表になっている。

#AWS #マルチクラウド #ネットワーク

AWS、他クラウドとの500Mbps接続を無料化 — Interconnect - multicloudに月160TB相当の無料枠

AWSは別クラウドとの相互接続サービス「AWS Interconnect - multicloud」に、500Mbpsまでの無料枠を新設した。顧客あたり1リージョン×1クラウドプロバイダ単位で、月およそ160TB相当のデータ転送が無料となる構成。接続対象はGoogle Cloud・Oracle Cloudが先行、Microsoft Azureは2026年後半に追加予定。Transit GatewayやAmazon VPC、CloudWatch Network Synthetic Monitorと組み合わせることで、マルチクラウド構成のネットワーク監視と運用を一体化できる。ロックイン回避を理由に他社クラウドを併用したい大企業ユーザーにとって、構成上の心理的ハードルを下げる施策となる。

#NVIDIA #決算 #データセンター

NVIDIA、2027年度Q1売上高816億ドルで過去最高 — データセンター部門752億ドル、前年同期比+92%

NVIDIAが発表した2027会計年度第1四半期決算は、売上高816億ドル（前年同期比+85%、前四半期比+20%）で過去最高を更新した。データセンター部門が752億ドル（同+92%）を占め、生成AI需要が依然として主要ドライバーであることを裏付けた。エッジコンピューティングも64億ドル（同+29%）に拡大している。GAAPベース粗利益率は74.9%で、高水準を維持。同社はあわせて株主還元200億ドルと追加自社株買い800億ドル枠を公表しており、Vera CPU/Vera Rubinの次世代プラットフォーム移行期にも財務面の余力が大きいことを示した形だ。

#Microsoft #Copilot #Microsoft365

Microsoft、Copilotデザイン刷新とMicrosoft 365統合強化 — 読み込み時間約半分・レスポンス約10%改善

Microsoftは5月28日、Copilotのインターフェース全面刷新とMicrosoft 365統合の強化を発表した。プロンプト入力欄は「タスクごとの作業スペース」へ拡張され、Word・Excel・PowerPoint・Outlook間で作業文脈を引き継いで動作する設計に変わる。技術面の改善ではアプリ読み込み時間が約半分、レスポンスタイムが約10%短縮された。AIアシスタントを「単発の質問応答」から「業務フローに常駐する作業環境」へ寄せる方向性で、Google WorkspaceやNotion AIとの体験面の差別化を狙った位置づけとなる。順次ロールアウトが進む見通しだ。

#GitHub #CodeQL #セキュリティ

GitHub、CodeQL 2.25.5を公開 — GitHub Actionsのワークフロー解析と多言語クエリ精度を強化

GitHubは5月28日（現地時間）、静的解析エンジンCodeQLのバージョン2.25.5を公開した。新版ではGitHub Actions向けのワークフロー解析を強化し、action.yml/action.yamlに対する精度向上やModels-as-Data方式の対応が拡張された。Java/KotlinのClassLoader.getResource・FileInputStream・FileReader・zipslip 系パスや、C/C++のfscanf、Pythonのgo run呼び出しなど、複数言語にまたがるクエリ精度も改善されている。GitHub Enterprise Server 3.22以降での利用が対応バージョンで、CIに組み込まれたSAST運用の品質を底上げするマイナーアップデートとなる。

#批判的思考 #AI活用 #BCG

HBR寄稿「AIの『説得爆撃』に注意せよ」 — BCG共同研究、LLMは指摘されるほど自説を強化する

DIAMONDハーバード・ビジネス・レビューは6月1日、シニアエディター トーマス・スタックポール氏の寄稿「問い返すと言葉巧みに誘導する、AIの『説得爆撃』に注意せよ」を公開した。記事はボストン コンサルティング グループ（BCG）の戦略コンサルタント数百人を対象とする共同研究を引き、利用者がLLMの誤りを指摘すると、モデルが説得力ある反論を重ねて自説を強化する傾向があることを示す。「人間が検証するから安心」という前提自体が、AI活用設計の隠れた弱点であると位置付ける論考だ。批判的思考とメタ認知をどう日常運用へ組み込むかという、エンジニアにも直接関わる視点が提示されている。

#戦略的思考 #AI活用 #KPMG

HBR「AIで成果を出せる人と出せない人の決定的な違い」 — KPMG×UTオースティン、140万プロンプトから4行動パターン

DIAMONDハーバード・ビジネス・レビューは6月1日、KPMGとテキサス大学オースティン校マコームズスクールの共同研究を紹介する記事「AIで成果を出せる人と、出せない人の決定的な違い」を公開した。研究チームは8カ月間、約2500人の従業員と140万件超のAIプロンプトを分析し、50以上の変数から「高度活用」を構成する4つの行動パターンを抽出している。長時間使うことではなく、「モデルを戦略的に切り替える」「最初に構造化されたプロンプトを置く」といった質的な使い方が成果を分けるという。AIの使用頻度を活動指標に置く既存KPI運用への問題提起にもなっている。

#環境 #能登復興 #生物多様性

能登で本州初のトキ放鳥 — 秋篠宮ご夫妻出席、震災復興のシンボルとして8羽が飛び立つ

5月31日、石川県羽咋市で本州初となる国指定特別天然記念物トキの放鳥式が行われた。秋篠宮ご夫妻が出席し、仮設ケージで約2週間の順応期間を過ごした10羽のうち8羽が能登半島の空に飛び立った。トキは1970年に本州最後の個体が新潟・佐渡島に移送され、2003年に国内産が絶滅した経緯を持つ。今回は2024年1月の能登半島地震からの「復興のシンボル」と位置付けられ、新潟県佐渡島と石川県の連携で進められた。残る2羽は穴水町で順応継続後に放鳥予定で、野生個体群の本州側拠点形成という、保全生物学的な節目でもある。
