#採用 #10X #新卒

10X、2027年卒プロダクトエンジニア採用開始 — Stailerの加盟チェーン130社超、TypeScript/GoスタックでEC基盤を担う

ネットスーパープラットフォーム「Stailer」を運営する10Xは、2027年卒向けプロダクトエンジニアの採用選考を開始した。生鮮食品のオンライン販売基盤として国内130社超の小売チェーンが加盟するStalierは、注文・在庫・配送の統合管理をTypeScript（Next.js/NestJS）とGoで実装するマルチテナントSaaS構成だ。技術職の選考は設計課題とコードレビューセッションを組み合わせたプログラムで、実務に近い問題設定となっている。2026年の改正フリーランス保護法施行を受け、小売業のデジタル化需要が加速しており、採用は通年受付・年次不問の形式で進められる。

#技術トレンド #MCP #API

Anthropic、MCP 2.0仕様を公開 — ツールストリーミング・マルチホップ認証・プログレス通知を追加

AnthropicはModel Context Protocol（MCP）バージョン2.0の仕様を6月27日に公開した。最大の追加機能は「Tool Streaming」で、ツール実行中の中間出力をリアルタイムにクライアントへ流せるようになった。これによりエージェントが長時間処理を行う際にも途中経過をUIに反映できるため、ユーザー体験が大きく改善される。また「マルチホップ認証」により複数のMCPサーバーをまたぐ権限委譲が仕様として標準化され、複雑なエージェントパイプラインのセキュアな実装が容易になった。プログレス通知APIと合わせ、GitHubやJetBrainsなど主要IDE・ツールベンダーがMCP 2.0への対応を表明している。

#技術トレンド #React #フロントエンド

React 20 RC リリース — ReactコンパイラとServer Componentsの統合完成、JSX transformの全面刷新

Metaは6月28日、React 20 Release Candidate（RC）を公開した。5月に発表されたBeta版から、ReactコンパイラとServer Componentsのシームレスな統合が完成した点が最大の強化ポイントだ。ReactコンパイラがServer ComponentsのRender出力を事前に最適化することで、クライアントへのバンドルサイズを抑えつつ再レンダリングを自動的に最小化する。JSX transformも刷新され、Preactとの互換性が高まったほか、`use()`フックがSuspenseとの連携をより直感的に記述できるよう改善された。Next.js・Remixの両フレームワークが同日中にRC対応をアナウンスしている。

Kyash、シリーズE 40億円を調達 — 金融インフラAPIエンジニアの増強とBtoB展開に本格投資

LayerX Tech Blog「GitHub Copilot導入1年後」— PR承認速度1.6倍・バグ検出率は有意差なし・コードレビュー指摘が質的に変化

LayerXのエンジニアリングブログが、GitHub Copilotを全社展開して1年後の実測データをまとめたレポートを公開した。PR承認までの平均リードタイムは導入前比1.6倍の速さに改善された一方、バグ検出率（本番障害件数で計測）には統計的有意差は見られなかった。顕著な変化として、コードレビューコメントの内容が「構文・タイポ指摘」から「設計方針・ドメイン整合性の議論」にシフトしており、LLMが自動修正できない上位レベルの問題に議論が集中する傾向が確認された。全エンジニアのAIコーディングツール費用は月次コストとして計上されており、1人あたりの費用対効果算定方法も詳細に記載されている。

#意思決定 #認知科学 #行動経済学

東洋経済オンライン「意思決定のコストを意識せよ」— 選択肢過多が判断力を消耗させる、Barry Schwartz「選択のパラドックス」の実践応用

東洋経済オンラインは、心理学者Barry Schwartzの「選択のパラドックス（The Paradox of Choice）」理論をAI時代の文脈で再解釈した論考を掲載した。AIがオプションを無限に生成できる環境ほど、選択肢を絞る「意思決定コスト管理」が人間のパフォーマンスに直結するという論点が主軸だ。脳が1日に下せる判断の総量には限りがあるため、AIによる選択肢過多が逆に判断精度を下げるリスクがある。対策として、判断ルールをあらかじめ「決定木」として言語化しておき、AIの選択肢を入力前にフィルタリングする方法が提案されている。「決める前に何を除外するか」を設計するメタ判断の重要性が説かれる内容だ。

#成人発達理論 #メタ認知 #思考フレームワーク

ダイヤモンドオンライン「成人発達理論の実践入門」— ケガン理論が解く「なぜ人は変われないのか」、垂直的成長の3段階

ダイヤモンドオンラインが、ハーバード大学教育学者ロバート・ケガンの成人発達理論をエンジニア・ビジネスパーソン向けに解説する連載を開始した。成人発達理論は「知識・スキルの量的拡大（水平的成長）」と「思考の枠組み自体が更新される（垂直的成長）」を区別するフレームワークで、なぜ経験を積んでも本質的な思考の変容が起きにくいかを説明する。ケガン理論は「社会化された心」→「自己主導する心」→「自己変容する心」の3段階を定義し、多くのビジネスパーソンが第2段階にとどまることを示す。AIに答えを委ねやすい現在の環境では第2段階の停滞が深まりやすく、第3段階への移行を意識した内省習慣が差別化につながるという論点が展開される。

#文化 #言語 #認知

中央公論 2026年7月号「書くことと考えること」— 村田沙耶香と認知科学者・今井むつみが語る言語化の限界と可能性

中央公論2026年7月号の特集「書くことと考えること」では、作家・村田沙耶香と認知科学者・今井むつみ（慶應義塾大学教授）が対談形式で「言語化することで思考が明確になる一方で、言語の型に思考が引きずられてしまう」矛盾を議論している。今井むつみは研究知見として、人は母語の文法構造が違うだけで事象の因果関係の把握方法が変わることを指摘する。村田は小説執筆を通じて「言語が思考を先取りしてしまう瞬間」を経験として語り、言語化しすぎることで思考の裾野が狭まるリスクを示している。AIが自然言語でアウトプットを生成する時代に、言語と思考の関係を改めて問い直す論考だ。

#国際 #環境 #都市設計

ニューズウィーク日本版 7月1日号「世界の都市再生2026」— ロッテルダムの浮体式住宅が気候変動時代の都市設計モデルに

ニューズウィーク日本版2026年7月1日号が特集「世界の都市再生2026」を掲載している。オランダ・ロッテルダムで海面上昇対応として建設された浮体式住宅地区「Waterbuurt」の現状と、世界の沿岸都市が同モデルを採用する動きが詳報されている。同地区は水上に固定された住宅・オフィス・公園で構成され、水位変動に応じて建物ごと上下する設計だ。気候変動適応型インフラとして、バングラデシュやミャンマーの沿岸部でも小規模実装が始まっており、土木・建築・気候工学が交差する現場として注目を集めている。技術が都市空間のあり方を変えてきた歴史を、建築の視点から理解する契機になる特集だ。
