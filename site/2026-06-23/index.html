#セキュリティ #AWS #AIエージェント

AWS、コード・インフラ構成・ビジネスコンテキストを統合解析してシステム全体の脆弱性を推論する「AWS Continuum」発表 — 特定のAIモデルに依存しない設計

AWSはコードのスキャンだけでなく、インフラ設定・権限設定・ネットワークトポロジー・ビジネスコンテキストまでを一括で解析し、システム全体の脆弱性を推論するAIセキュリティサービス「AWS Continuum」を発表した。既存のAmazon InspectorとCodeGuruを統合・進化させた後継に位置付けられ、発見・優先順位付け・検証・修復の4フェーズをマシンスピードで実行する。特定のAIモデルに依存しない「モデル非依存アーキテクチャ」を採用し、各領域で最も得意なモデルを自動選択する設計が将来のモデル更新にも柔軟に対応できるようにしている。設計書とソースコードから脅威モデルを自動生成する「Threat Modeling」のプレビュー提供も始まっており、セキュリティを内製化するチームの設計議論への活用が期待されている。

#GitHub #CopilotSDK #AIコーディング

GitHub Copilot SDK、TypeScript/Python/Go/.NET/Rust/Javaの6言語で一般提供開始 — CIパイプラインや自動化ワークフローへの組み込みが本番利用可能に

GitHubはMicrosoft Build 2026で発表したGitHub Copilot SDKをTypeScript・Python・Go・.NET・Rust・Javaの6言語で一般提供（GA）開始した。CI/CDパイプラインへの組み込みやIssueトリアージ・PRレビューの自動化など、エンタープライズ規模のワークフロー自動化にCopilotのコード生成・解析能力を直接使えるようになる。SDKはAnthropic Claude・OpenAI GPTを含む複数のモデルを切り替えられる統一インターフェースを持ち、GitHub Copilot Pro+/Enterpriseユーザーが対象となる。GitHub Copilotは2026年6月から従量課金制（GitHub AI Credits）に移行しており、SDK利用のコスト管理と使用量モニタリングが実装上の重要な設計事項となっている。

#採用 #ZOZO #インターン

ZOZO、2027年卒対象の就業型インターンシップ本格化 — エンジニア職に加えPM職を新設、本選考直結型で受付中

ZOZOは2027年卒を対象とした開発部門の就業型インターンシップの受付を開始している。従来のエンジニア職に加え、今回からプロダクトマネージャー（PM）職の募集も新設され、ZOZOTOWNをはじめとする実際のプロダクト開発チームに参加する設計となっている。インターンシップは本選考直結型として位置付けられており、参加者はプロダクトの設計・開発・改善サイクルに直接関与する。ZOZOは独自の計測技術（ZOZOGLASS・ZOZOMAT）とファッションECの購買データを組み合わせたサービス開発を展開しており、バックエンド・フロントエンド・機械学習の各領域でプロダクト思考を持つエンジニアを求めている。

#採用 #DeNA #AIジェネラリスト

DeNA 2027年度「AIジェネラリスト」コース選考本格化 — 標準年収600万円スタート、AIを実装で使うスキルを重視

DeNAは2027年度新卒採用でAIを実装から活用できる人材を対象とした「AIジェネラリスト」コースの選考を本格化している。標準年収600万円（月給46.5万円）スタートで、配属先ではゲーム・スポーツ・ヘルスケアにまたがる各事業のプロダクト開発チームでAI機能の設計から実装まで担当する。短期就業型インターン（夏以降）から早期選考へつながるフローが設計されており、8月以降に1〜3か月の長期インターンを経て本選考に進むルートも用意されている。選考では生成AI・強化学習・最適化アルゴリズムなど複数AI技術の実装経験を評価軸の一つとしており、「AIを使って事業成果を出す」エンジニアを求めるコンセプトが特徴的だ。

#Anthropic #Claude #AgentSDK

AnthropicのClaude Agent SDK課金体系変更、2026年6月15日の施行当日に緊急停止 — 開発者コミュニティの混乱受け

Anthropicは約1か月前から予告していた「Claude Agent SDK利用を対話利用と別クレジットプールで管理する」課金変更を、2026年6月15日の施行予定当日に緊急で一時停止した。変更対象だったのはAgent SDK（Python/TypeScript）・claude -pコマンド・Claude Code GitHub Actions・認証連携のサードパーティアプリの4種類で、プランごとに月額料金相当のクレジットが付与される設計だった。施行当日に変更撤回が発表され、現在もAgent SDK利用は通常のサブスクリプション利用制限から消費される状態が継続している。今後の再発表時期は未定で、ClaudeのAgent機能を業務パイプラインに組み込んでいるチームにとって、コスト試算と利用制限の設計を事前に整備しておく重要性を示す事例として業界で議論が続いている。

#判断力 #HBR #リーダーシップ

DIAMOND ハーバード・ビジネス・レビュー「AI時代の判断力」— 生成AIは既存の判断力を増幅するが判断力の欠如は補完しない

DIAMOND ハーバード・ビジネス・レビューに掲載されたリーダーシップ研究者の論考は、生成AIの普及が組織内の判断力格差を拡大させているという逆説を提示している。AI導入後に経験の浅い社員のパフォーマンスが底上げされるという仮説とは逆に、判断力のある人材とない人材のパフォーマンス差がAI使用後にむしろ拡大したという観察が示されている。AIが流暢な回答を生成することで「判断している風」の振る舞いを低コストで演じられるようになり、本来の判断力を育てる機会と失敗経験から学ぶプロセスが失われるという分析だ。解決策として筆者はAIを使わずに問いと向き合う「判断練習の場」を組織に意図的に設けることを提案しており、ルールだけでは動けない状況での意思決定能力こそがAI時代の競争力の源泉だと論じる。

FIFAワールドカップ2026北中米大会グループJ第2節、アルゼンチン対オーストリア戦でリオネル・メッシが2得点を挙げ、W杯通算18ゴールで歴代最多得点記録を単独更新した。前半38分に左からのクロスに合わせた左足ボレーで先制し、後半アディショナルタイムにゴール前の混戦を押し込んで追加点を挙げ、1970年メキシコ大会のジャイルジーニョ以来56年ぶりとなるW杯6試合連続得点も記録した。アルゼンチンは2-0の完封勝利でグループ首位に立ち、2試合で決勝トーナメント進出が確定した。北中米大会に参加した38歳のメッシが前代未聞の記録更新を大会第2節で達成したことは、北米を主会場とする今大会最大の話題の一つとなっている。
