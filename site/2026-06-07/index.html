#Node.js #OSSリリース #ランタイム

Node.js 26.3.0公開 — macOSユニバーサルバイナリの今後は不透明、Buffer.poolSizeは64KiBへ拡張

Node.js プロジェクトは6月1日付で Node.js 26.3.0 を公開した、とCodeZineが6月4日に報じた。Apple Silicon と Intel の両アーキテクチャを単一バイナリでサポートしてきた macOS ユニバーサルバイナリは、Apple が Intel サポートを段階的に終了する中で今後の配布方針が不透明になっており、当面はデュアルアーキテクチャ対応を継続する。機能面では Buffer.poolSize のデフォルトが 64KiB に拡張、ルート証明書が NSS 3.123.1 へ更新、http モジュールの検証カスタマイズ用オプションが追加、inspector の coverage API が強化された。Web系バックエンドの足回りに直結する更新で、TypeScript / フロント側ツールチェーンの動作前提にも影響が及ぶ。

#Django #セキュリティ #Python

Django 6.0.6 / 5.2.15が複数の脆弱性を修正してリリース — 5件のCVE、Cookie名前空間衝突や暗号化メール送信問題に対応

Djangoプロジェクトは6月6日、セキュリティリリース Django 6.0.6 と 5.2.15 を公開したとCodeZineが伝えた。修正された脆弱性は計5件で、Cookie名前空間の衝突 (CVE-2026-6873)、メールが暗号化されずに送信されるリスク (CVE-2026-7666)、キャッシュ制御ヘッダの扱い (CVE-2026-8404, CVE-2026-48587)、Vary ヘッダ欠落による私的データ露出 (CVE-2026-35193) を含む。すべてのバージョンで影響範囲が広く、利用ユーザーには速やかなアップグレードが推奨されている。Cookie / キャッシュ / メール送信といったHTTPレイヤの基本に複数の修正が集中しており、Web系プロダクトのバックエンド衛生に直結するリリースとなった。

AIコスト高騰で中国DeepSeekへの「乗り換え」続出か — 米Ramp調査でV4-Proが入力1Mトークン$0.435と最安水準

米決済プラットフォーム Ramp の5万社以上を対象にしたAI支出調査で、中国 DeepSeek が成長率カテゴリで首位を取った、とITmediaが6月5日に報じた。価格差が乗り換えを牽引しており、DeepSeek V4-Pro は1Mトークンあたり入力$0.435 / 出力$0.87、対する OpenAI は $5 / $30、Anthropic は $5 / $25。さらに DeepSeek は5月23日に75%引き価格を恒久化した。アナリストは「2025年1月の旋風と同パターンで持続性を過大評価しない方がよい」と慎重姿勢だが、Web系プロダクトの本番AIコスト構造に「中国モデル前提」の選択肢が現実味を帯びてきた局面となっている。

#Supabase #シリーズF #PostgreSQL

Supabase、評価額105億ドルで5億ドル(約800億円)調達 — シリーズFでGIC主導、データベース数は前年比6倍に

PostgreSQLベースのBaaS Supabase が、評価額105億ドルで5億ドル(約800億円)のシリーズF資金調達を実施したと6月5日に発表した。リードはシンガポール政府投資公社 GIC、ほかにAccel、Y Combinator、Craft、Felicis、Coatue、Stripe、Salesforce Ventures が参加。顧客数は25万社を超え、ユーザーベースは1年で2倍、データベース数は前年比 600%増と、AIエージェントによる大量プロビジョニングが成長を押し上げた。調達資金はエージェント型AIインフラ領域での主導権強化に投じられ、同時に単一インスタンス制約を解消するPostgreSQLスケーリング基盤 Multigres のプレビューも発表された。「BaaS × AIエージェント」が大規模商用領域に踏み込んだことを示すスタック転換の事例となる。

#ドラッカー #未来思考 #DHBR

DHBR「ドラッカーが説いた、未来をつくる2つの方法」 — 「すでに起こった未来」と「来るべき未来を出現させる」

ピーター・ドラッカーは、未来を予測するのではなく未来を「つくる」ための方法を 2つ提示している、とDHBRが6月5日に解説する。1つ目は「すでに起こった未来」 — 社会・経済・知識の不連続点はすでに進行しており、そのリードタイムを活用して動く。2つ目は「来るべき未来を出現させる」 — 何が現れるべきかを構想し、自ら実現させていく姿勢。「未来を予測するだけでは問題をまねくだけである」というドラッカーの言葉は、AI で予測の解像度ばかり上がる時代に、「どの不連続を今動かすか」という問題定義の力そのものに焦点を戻させる。論点の組み立て方を、「予測の精度を上げる」から「不連続を選び取る」へとずらすための補助線になる論考。

#理論物理 #弦理論 #Caltech

弦理論は数個の物理的前提から自然に出現すると判明 — Caltechがたった4つの仮定から導出、PRLに掲載予定

米カリフォルニア工科大学 (Caltech) の Clifford Cheung らのチームが、「ひもがある」と最初に仮定せずに、4つの基本的な物理学的前提だけから弦理論の方程式が数学的に必然として導出されることを示した、とナゾロジーが6月5日に報じた。論文タイトルは「Strings from almost nothing」、『Physical Review Letters』に掲載予定。これまでは「便利だが任意」と見られてきた弦理論が、「素朴な前提を満たすにはこの形しかない」という意味で、物理学の基礎的言語の候補に格上げされる可能性がある。「最初に何を仮定するか」だけで結論の形がほぼ決まる、という構造そのものが第一原理思考の素材として読み取れる事例だ。

#植物科学 #シロイヌナズナ #名古屋大学

名古屋大、シロイヌナズナの種子サイズを決める「栄養の門」を発見 — 篩部末端ゲートが受精後の栄養流入量を制御

名古屋大学 生命農学研究科のチームが、世界で最もよく研究されているモデル植物 シロイヌナズナ (Arabidopsis thaliana) の種子サイズを決めている「栄養の門」を発見した、とナゾロジーが6月7日に報じた。胚乳に栄養を送り込む師管末端 (篩部末端) に「ゲート」のような構造があり、受精成功後に種子へ流入する糖・アミノ酸の量を能動的に制御していることが判明。論文は『Current Biology』に掲載された。このゲートを操作できれば、大きさ・栄養価をコントロールした作物の育種が可能になり、コメ・小麦・大豆など主要穀物の収量設計への応用が見込まれる。食料安全保障・気候適応の文脈で、Web/IT外の研究フロンティアが一歩進んだ事例。
