BASE、2027年卒ソフトウェアエンジニア採用を開始 — EC×決済プロダクト実装経験を選考評価の軸に

BASEは2026年5月下旬、2027年卒ソフトウェアエンジニアの採用受付を開始した。ECプラットフォーム「BASE」と決済サービス「PAY.JP」の両プロダクトを軸に、TypeScript・Go・Kotlinいずれかの実装経験を選考評価の軸に置く形式を採る。夏季インターンシップは5日間のプロダクト開発課題を中心に構成され、インターン通過者は本選考に直結する。ECプラットフォームと決済インフラの両ドメインに横断して携わることができる国内事業会社として、Webアプリケーション開発に関心を持つ学生の間で認知度が高まっている。

#採用 #グリー #新卒

グリー、2027年卒エンジニア採用で「AIゲーム開発コース」を新設 — 採用目標を前年度比2倍に引き上げ

グリーは2026年5月、2027年卒エンジニア採用に「AIゲーム開発コース」を新設した。生成AIをゲームエンジン・NPC挙動・コンテンツ自動生成に組み込む開発に特化したコースで、採用目標人数を前年度比2倍に設定している。既存の「バックエンドエンジニアコース」と「インフラエンジニアコース」に加えた第3の選考ルートで、面接はLLM APIを使ったゲームプロトタイプ実装課題を軸に評価する形式を採る。2026年度から2027年度にかけて開発組織を積極拡大する計画のもと、AIゲーム開発エンジニアを通年で受け付けている。

#Go #技術トレンド #OSS

Go 1.27 RC1 公開 — iterパッケージ正式標準入り、GCストップザワールド時間を最大20%短縮

Googleは5月27日、Go 1.27のリリース候補1（RC1）を公開した。Go 1.23で実験的に導入されていた iter パッケージ（range-over-func）が正式標準ライブラリに昇格し、for range によるイテレータ記述が推奨パターンとして確立される。ガベージコレクタは並行スキャンの内部スケジューラを改良し、ストップ・ザ・ワールド（STW）時間をベンチマーク上で最大20%短縮した。ツールチェーンでは go test の並列化制御フラグが拡張され、大規模モノレポでの実行効率が向上している。正式リリースは2026年8月予定で、RC1の試用フィードバックを募集中だ。

#React #JavaScript #OSS

React 19.2リリース — Server Actionsが安定版APIに昇格、Suspenseのストリーミング挙動を改善

MetaのReactチームは5月27日にReact 19.2を公開した。19.0から実験的フラグ下に置かれていたServer Actionsが安定版APIとして昇格し、canaryフラグなしで利用できる。<Suspense>のストリーミング挙動について、ネストした境界が同時にフォールバックを表示する際に親が子を待つ深度を設定可能になり、ウォーターフォール問題の軽減に役立つ。React CompilerによるuseMemo・useCallbackの自動最適化と組み合わせた再レンダリング削減も強化された。既存の19.xプロジェクトはパッケージバージョンを上げるだけで利用可能で、破壊的変更はない。

#Cloudflare #AI #インフラ

Cloudflare Workers AI、推論ルーティングをGA — レイテンシを最大40%改善するリージョン自動振り分け

Cloudflareは5月28日、エッジAIサービス「Workers AI」においてリクエストを自動的に最短レイテンシのリージョンへ振り分ける「Global Inference Routing」を正式一般公開（GA）した。北米・欧州・アジア太平洋の各リージョンにGPUノードを分散配置しており、東京起点のリクエストでは大阪・シンガポールの自社エッジが使われることが多く、従来の単一リージョン指定と比べレイテンシを最大40%改善したとする社内ベンチマークが示された。Llama・Mistralなどオープンモデルのサーバーレス推論が対象で、APIキーひとつで利用できる。Cloudflare WorkersでAIを扱う個人開発やスタートアップのインフラ選定肢として国内エンジニアの間での採用実績が増えている。

#思考法 #問いの設計 #構造的思考

「問いの設計力」— 前提を解体してAIと差をつける構造的思考の実践

ダイヤモンド・オンラインが5月下旬に公開した記事では、生成AIが「答えを即座に出す」ツールとして定着した今、問い自体を構造化できる人材との差が広がっていると論じている。問いの設計には「前提の解体（What am I assuming?）」「反例の検証（What would disprove this?）」「レベル変更（What is the higher-order question?）」の3ステップが有効とされ、ソクラテス式問答法の現代的応用でもある。スタートアップの意思決定事例として、「売上を上げるにはどうする？」という問いを「その売上増加は何のためか？」と上位の問いに書き換える実践例が紹介されている。「問いの粒度を合わせること」が組織の意思決定品質を左右するという視点は、チームでのプロダクト開発や面接でも具体的に転用できる思考の鍛え方だ。

#行動経済学 #認知バイアス #習慣設計

行動経済学が解く「先延ばし」のメカニズム — 現在バイアスへの予防的設計が有効

東洋経済オンラインが掲載した行動経済学の解説記事では、先延ばしの根本原因を「現在バイアス（present bias）」——遠い将来の報酬を大きく割り引いて現在の快適さを優先する傾向——と特定している。これはカーネマンの「システム1（速い思考）」が長期メリットよりも短期のコストを強調するためで、意志の問題ではなく脳の設計上の特性だ。対策として提唱されるのは「プリコミットメント（pre-commitment）」——行動を実行する前に選択肢を事前に制約する仕組みを作ること。締切より1週間前に「仮提出版を見せる約束」を知人と設定することで、自分の意志力に頼らず着手を強制できる。行動経済学の知見を自分の行動設計に組み込む実践として、理論だけでなく行動の変化につながる内容だ。

#宇宙 #科学 #JAXA

JAXAきぼう実験棟で無重力コムギ3世代連続栽培に成功 — 宇宙農業実現へ重要な一歩

JAXAは5月28日、国際宇宙ステーション（ISS）日本実験棟「きぼう」内で実施した植物育成実験において、コムギを無重力環境下で3世代にわたって連続栽培することに成功したと発表した。2023年から開始した実験で、種から発芽・開花・結実のサイクルを地球外で完結させた形となる。宇宙農業は将来の月面・火星探査で乗組員への食料自給の鍵とされており、今回の成果は作物が宇宙環境に継続的に適応できることを示す重要な一歩となる。無重力下でも根が走光性で成長方向を補正することも確認され、成果はNPJ Microgravityに掲載された。

#将棋 #AI #文化

棋士×AI共同研究の棋譜を日本将棋連盟が初公開 — 藤井聡太七冠「AIが問いの発生源になった」

日本将棋連盟は5月29日、現役棋士と生成AIが協力して研究した棋譜セットを初めて公式に公開した。藤井聡太七冠が担当した棋譜では、AIが提示した候補手を起点に人間がその意図を逆解析する手法を用い、「AIの提案を単に採用するのではなく、なぜその手が有力なのかを自分で考え直す工程で発想が広がった」と語っている。将棋の序盤研究ではすでにAIが主流の分析ツールだが、今回の取り組みはAIを「答えの提供者」ではなく「問いの発生源」として活用する実践例として注目された。人間とAIの役割分担の在り方を将棋という知的作業の現場から示す事例で、ソフトウェア開発の現場でAIをどう活用するかという問いを具体的に照らし出している。
