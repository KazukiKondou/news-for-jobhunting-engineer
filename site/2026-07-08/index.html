ログラス、2027卒エンジニア採用を通年化 — 経営管理SaaSのバックエンド/フロントエンド5名同時公募

ログラスは2026年7月、経営管理クラウド「Loglass」の開発を担うプロダクトエンジニアを2027年卒向けに通年採用すると発表した。バックエンド（Go・Kotlin）とフロントエンド（TypeScript・React）の2ロールで合計5名の採用枠を設けており、両ロールへの同時応募も可能としている。選考は書類審査・コーディング課題・技術面接・最終面接の4ステップで、コーディング課題は競技プログラミング形式ではなくプロダクト設計を問う独自問題を採用している。インターン参加者向けには早期選考ルートが用意されており、今夏開催予定のサマーインターン枠も同サイトで受け付けている。

Chatwork 2027卒エンジニア夏季インターン「CW Tech Campus」応募開始 — 5日間で実際のSaaS機能をリリースまで担当

Chatworkは2027年卒向け夏季インターンプログラム「CW Tech Campus」の応募受付を7月8日に開始した。期間は5日間で、ビジネスチャット「Chatwork」の実機能をチームで設計・実装しステージング環境へリリースするまでの工程を体験できる実務型カリキュラムとなっている。主な技術スタックはScala・React・TypeScriptで、コード品質・設計力・チームでの協働力が評価の軸に据えられている。参加者には日額5,000円のインターン報酬と交通費・宿泊費が支給される。2025年の参加者のうち約40%が選考フローへ接続しており、本選考への直結ルートとして機能している。

#TailwindCSS #CSS #フロントエンド

Tailwind CSS v4.1 正式リリース — CSS Cascadeレイヤー対応と新しい@starting-styleアニメーション命令を追加

Tailwind CSS v4.1が7月7日に正式リリースされた。主な追加機能はCSS Cascade Layersへのネイティブ対応で、@layerディレクティブとの相互運用性が向上し既存のCSSライブラリとの統合コストが下がった。またCSS @starting-styleに対応したことで、DOM挿入時のフェードインなどの表示トランジションをユーティリティクラスだけで定義できるようになった。パフォーマンス面ではv4.0から引き継いだRustベースのビルドエンジンを踏襲しており、中規模プロジェクトのコールドビルドが平均50ms以下に収まる点は変わらない。既存のv4.0プロジェクトからの移行は設定変更なしで対応可能とされている。

#GitHub Actions #CI/CD #macOS

GitHub Actions、macOS M4ホステッドランナーを全プランに正式GA — Apple Silicon CIビルドが最大2.5倍高速化

GitHubは7月8日、macOS M4搭載のホステッドランナー（macos-m4-latest）を全プランのユーザーに正式提供（GA）した。テクニカルプレビュー期間を経て、Freeプランを含むすべてのアカウントから追加設定なしで利用できる。GitHubが公表したベンチマークでは、同スペックのx86_64 macランナーと比較してXcodeによるフルビルドが最大2.5倍高速化しており、iOSアプリのCI自動化で効果が顕著だ。料金はM2ランナーより約15%高いが、ビルド時間の短縮によりCI実行コストを実質削減できるケースが多いとGitHubは説明している。

#認知バイアス #意思決定 #DHBR

「確証バイアスとエコーチェンバー」— 信じたい情報だけを選ぶ認知の罠と構造的な破り方

ダイヤモンド・ハーバード・ビジネス・レビュー（DHBR）の特集が確証バイアスのメカニズムと組織的意思決定への影響を解説している。確証バイアスとは自分の信念を支持する情報を優先的に収集し、反証となる情報を無意識に退ける傾向で、カーネマンが「システム1（速い思考）」と呼ぶ自動処理が常に作動することで生じる。同質性の高いチームほどこの罠にはまりやすく、メンバーが互いの意見を確認し合う「エコーチェンバー」に陥ると自己強化が加速する。破り方として記事が紹介するのは「反証探索の義務付け（Disconfirmation Protocol）」と「事前検死分析（Premortem Analysis）」の2手法で、どちらも「この判断が間違いだったとしたら何が起きているか」を事前に問うことが核心となる。

#成人発達理論 #思考力 #成長

「垂直的成長」とは何か — AI時代に問われる認識の枠組みそのものの変容

東洋経済オンラインが成人発達理論における「垂直的成長」の概念を解説している。ロバート・キーガンの理論に基づくこの概念は、知識・スキルの蓄積である「水平的成長」とは異なり、「ものの見方・判断の枠組み」そのものが変容する深い成長を指す。AIが知識検索と情報処理を代替できる時代に差別化となるのはこの垂直的成長であり、単なる技術習得に止まらない認識の高度化が求められるという主張が記事の軸となっている。垂直的成長を促す実践として著者は「自分が確信している前提と逆の立場をあえて弁護する」「自分の反応に対して『なぜそう感じたか』を問い直す」という2つの習慣を挙げており、どちらも自己の思考プロセスを俯瞰するメタ認知の訓練となる。

#宇宙 #ispace #月面探査

ispace、月面着陸機「RESILIENCE」ミッション3の打ち上げを2027年2月に確定

宇宙ベンチャーのispaceは7月8日、HAKUTO-Rシリーズ3号機「RESILIENCE」の打ち上げ予定を2027年2月に確定したと発表した。2023年4月のミッション1は月面降下中の高度推定エラーが原因で着陸直前に失敗し、2025年6月のミッション2は月面到達には成功したものの着陸後に通信が途絶えた経緯がある。ミッション3では制御アルゴリズムの大幅な改良に加え、着陸地点を平坦な溶岩台地「ラクス・モルティス（Lacus Mortis）」に変更し、地形認識ナビゲーションも新搭載する。ispaceは宇宙×ソフトウェアの両面でエンジニアを積極採用中で、国際チームでのミッションクリティカルな開発環境が特徴とされている。

#代替タンパク #農業 #食糧問題

「食べるコオロギ」から「培養魚肉」へ — 代替タンパク市場の主役交代と食糧安保の論点

日経が特集する代替タンパク市場の動向によると、2020年代前半に注目された昆虫食（コオロギ等）は消費者の受容率の低さから市場拡大が停滞し、代わりに細胞培養魚肉と発酵タンパクが急成長している。魚の筋肉細胞を培養する「セルフィッシュ」技術は水産物の乱獲問題と連動する形でESG投資の対象となり、日本でも複数のスタートアップが参入している。農水省の有識者会議では、大豆輸出国への依存リスクを低減する観点から国産代替タンパクを食糧安保戦略に組み込む議論が始まった。代替タンパクを選ぶ動機が「倫理・環境」から「コスト・利便性」へ移行しつつあることが普及の鍵とされている。
